# Automated Test via Cypress Setup Guide

## 1. Install Node.js
- Download and install Node.js (LTS version recommended) from [https://nodejs.org](https://nodejs.org).

## 2. Install Cypress
- Run "npm install" to enable cypress locally

## 3. Run Cypress
- Run "npx cypress open"
- In the "e2e Testing" mode click on the "wizardFlow" to execute the test
